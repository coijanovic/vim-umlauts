# vim-umlauts

This is *vim-umlauts*, a plugin to make writing german text with a qwerty keyboard easier.

Calling `:Umlauts` will make the following replacements in the current buffer:

| Before | After |
|--------|-------|
| :a     | ä     |
| :A     | Ä     |
| :o     | ö     |
| :O     | Ö     |
| :u     | ü     |
| :U     | Ü     |
| :s     | ß     |
